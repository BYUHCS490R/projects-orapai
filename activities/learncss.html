Explain the difference between inline, embedded, and external CSS.

Inline css is a style within a tag, Embedded css is a style in the head, can be placed just below the title, external css is a separate file of style that can be used by multiple html files.

Explain why you should only use external CSS.

Using external CSS can help keep the style the same throughout all the webpages you are designing. This will also save time.

What is a selector used for?

Selector is use to specify where in the page and which tag you want the style to be applied.
