Meaning in Your Life

What is the root? What is the relationship between folder and file names and the path?

The root is the starting point of a website. The path shows how to get from the root through folders to a specific file, so folders and file names create the steps in that “address.”

What is a default document?

A default document is the page that automatically loads when a user visits a website, often called index.html. This helps websites open a starting page without the visitor needing to type the file name.

Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link uses the entire URL including domain, while a relative link points to a file based on the current folders location. Relative links are helpful when working on the same website because they make it easier to move files without breaking the links.

HTML Tags

List the 3 rules with relative links.

First, if the file is in the same folder, just use the filename. Second, if the file is in a subfolder, include the folder name and a slash before the filename. Third, if the file is in a higher-level folder, use ../ to go up a level.

Do links always need to be inside a NAV tag?

No, links do not always have to be inside a NAV tag. The NAV tag is best for main navigation menus, but links can also go in the footer, main content, or even inside a paragraph if they connect to useful resources.
