ASCII and Unicode

American Standard Code for Information Interchange is use to represent text in a standardized way. It has a 7 bit character encoding standard which can represent 128 unique characters

Unicode is essential for international software because it supports virtually all characters. ACSII that is limited to characters for theh English alphabet and basic symbols.

What is UTF-8

The most common character encoding system for the internet. Allows for the representation of almost every character and symbol from any writing system in the world.

URL

Uniform Resource Locator is the unique web addresss for a specific page on the internet.

Domain Name

Easy to use web address that points to a website's IP adress on teh internet.

HTTP

Hypertext Transfer Protocol is the foundational internet protocol for transmitting information

HTTPS

Hypertext Transfer Protocol Secure is an encrypted version of HTTP to secure data transfer between browser and server

IP Address

Internet Protocol Address is a unique identifier for devices on a network

DNS

Domain Name System is the "phonebook" of the internet. It translates domain names into Ip addresses that computers can use to connect.

W3C Recommendations

World Wide Web Consortium recommendation define how web technologies like HTML, CSS, XML should work to ensure compatibility across platforms and browsers.
