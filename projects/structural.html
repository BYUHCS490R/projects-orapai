Structural tags

header, nav, main, footer

These structural tags organize a webpage so it's easier to read and be more accessible.

What goes in the header?

The header has all the important links for the page as well as the site title or logo.

What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

The nav holds navigation menus (it can be inside or outside the header and may include links to other websites)

What goes in Main?

The main section holds the pages's main content

What goes in the Footer?

Footer has the contact info, copyright, or related links.

HTML Validation

Checks the code for errors and ensures it follows web standaards.
