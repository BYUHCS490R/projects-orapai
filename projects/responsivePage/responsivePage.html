Why is the viewport meta tag important?

Viewport meta tag makes web pages look good on all devices by controlling how a page is scaled and displayed on different screen sizes, especially mobile phones. Without it, website might appear zoomed out or too small on mobile screens.

How can you adjust the CSS for printing?

By using the media print rule in your stylesheet, you can adjust the CSS for printing.

When would you want to adjust the CSS for printing?

When you expect users to print pages, you would want to adjust the CSS so the printed pages would look cleaner by removing unnecessary elements like nav links or buttons.
