Why do we need a viewport meta tag?

We need the viewport meta tag so our website displays correctly on all devices, especially phones and tablets. It makes the page responsive by telling the browser how to scale and fit the content to the screen size.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that’s written last in the CSS file takes priority because CSS reads from top to bottom — the most recent rule overrides the earlier one if they target the same element.

How do you change the CSS for a web page displayed on a phone?

You can change the CSS for a phone by using media queries, like @media (max-width: 600px) { ... }, which lets you apply different styles when the screen size is smaller than a certain width.
