What are different ways to style your form?

You can style forms using inline CSS, external CSS. External CSS is best for consistent, reusable, and responsive form design

How does flex help for both Desktop and Mobile phones?

Flexbox helps by automatically adjusting the layout, showing elements side-by-side on desktops and stacking them vertically on mobile screens. It makes forms responisve and easy to use across different devices.
