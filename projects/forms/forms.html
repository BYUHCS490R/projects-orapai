The form tag has an "action" attribute. What does it do?

The action attribute tells the browser where to send the form data when the user submits the form

The form tag has a "method" attribute. What does it do?

The method attribute specifies how the form data is sent to the server.

What is the difference between a "POST" and a "GET"?

GET sends data appended to the URL which is visible, it's when it read something. POST sends data in the request body which is hidden in the URL, it's when we change, add, update, or delete something
