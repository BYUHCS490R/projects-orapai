What is JavaScript? How did it get its name?

JavaScript is a core web technology used to make web pages interactive and dynamic, working alongside HTML and CSS. Its initial name, LiveScript, was changed to JavaScript primarily as a marketing move to associate it with the popular Java language at the time.

What are the two ways you can run JavaScript?

The two main ways are running it client-side inside a web browser, where the browser's engine executes the code embedded in an HTML file. The other is running it server-side or standalone using a runtime environment like [Node.js]

What was the most popular front-end js library?

The most popular historical front-end library was jQuery, which greatly simplified complex client-side tasks like DOM manipulation and AJAX for a wide range of websites.

What is the most popular front-end js library today?

Today, the most popular choice is React, a library developed by Meta (Facebook) for efficiently building modern, component-based user interfaces.
