When do you use a table?

When you have data to present.

When do you not use a table?

You dont use a table for layout or design purposes, for example, to position images or text on a webpage. Thats what CSS (like Flexbox or Grid) is for. Tables should only be used for tabular data.

When do you need to do extra work for table accessibility?

You need to add accessibility features when your table is complex (like when it has multiple headers or merged cells). This includes adding proper th tags, scope attributes, and sometimes aria labels to help screen readers interpret the data correctly.
